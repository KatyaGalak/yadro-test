# Эмулятор ленточного устройства и сортировщик

## Системные требования

* **C++20**
* **CMake** версии 3.12 или выше
* **GoogleTest** (загрузка осуществялется автоматически во время сборки)

## Инструкция по сборке и запуску

### 1. Конфигурация проекта
```cmake -S . -B build```
### 2. Компиляция
```cmake --build build```
### 3. Запуск тестов
```ctest --test-dir build --output-on-failure```
